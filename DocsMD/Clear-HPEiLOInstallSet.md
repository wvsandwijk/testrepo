# Clear-HPEiLOInstallSet

## SYNOPSIS
Clears the Install set.

## SYNTAX

```
Clear-HPEiLOInstallSet [-Connection] <Connection[]> [-IncludeRecoverySet] [-OutputType <String>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The Clear-HPEiLOInstallSet cmdlet clears the Install set.
This cmdlet is supported only on iLO 5 servers.

## EXAMPLES

### EXAMPLE 1
```
PS C:\> $connection = Connect-HPEiLO 10.20.30.2 -Username admin -Password admin123

PS C:\> ,$connection | Clear-HPEiLOInstallSet
```

This cmdlet clears the Install set on iLO 5 with the connection object passed as pipeline input.

### EXAMPLE 2
```
PS C:\> $connection = Connect-HPEiLO 10.20.30.1,10.20.30.2 -Username admin -Password admin123

PS C:\> Clear-HPEiLOInstallSet -IncludeRecoverySet No -Connection $connection
```

This cmdlet clears the Installation task queue on iLO 5 with an array of connection objects passed as named parameter.
